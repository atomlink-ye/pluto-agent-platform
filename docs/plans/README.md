## Format

Each plan is a markdown file. Use this template:

```markdown
# Plan: <title>

## Goal
<One sentence>

## Scope
- Included
- Excluded

## Status
- [ ] Not started
- [ ] In progress
- [ ] Blocked
- [ ] Complete

## Tasks
1. <task>
2. <task>

## Dependencies
- <depends on>

## Notes
<Misc>
```
